FASTEST DEPLOYMENT (5 minutes):
================================

OPTION 1: Netlify Drop (Easiest)
---------------------------------
1. Go to https://app.netlify.com/drop
2. Extract shame-to-flame-static.tar.gz
3. Drag the "out" folder onto the Netlify Drop zone
4. Done! Your site is live with a free URL
5. (Optional) Add your custom domain in settings

OPTION 2: Vercel (Also Easy)
-----------------------------
1. Go to https://vercel.com
2. Sign in with GitHub
3. Click "Add New Project"
4. Upload the folder or connect GitHub repo
5. Done! Auto-deploys with every update

OPTION 3: GitHub Pages (Free Hosting)
--------------------------------------
1. Create a new GitHub repository
2. Upload the contents of the "out" folder
3. Go to Settings â†’ Pages
